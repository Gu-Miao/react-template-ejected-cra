# React Template Ejected CRA

简体中文 | [English](./README.md)

这是一个 `React` 开发模板，基于 `CRA@4`，我将项目 `eject`，并对它进行了一些修改来增强和补充项目的一些功能：

- Prettier 和 eslint 支持。
- Antd 和按需引入。
- CHANGELOG 模板。
- Editor config。
- 独立的 babel 和 eslint 配置文件。
- 打包分析和速度测试的 webpack 插件。
- Webpack 别名。
- Less 支持。
- 删除冗余 CSS。
- 简单的配置式路由和缓存路由。
